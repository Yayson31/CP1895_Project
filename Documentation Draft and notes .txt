rules of the game : Dice Duel

UML diagram will help show visual - this was just how I picture it 

player vs house(computer) dice rolling game 

each round player and computer roll six sided die

highest roll wins the round = score is track for each round won per player 

first to win 10 rounds wins the game

round rules
highest roll wins 1 point 
only one roll for each player per round 
any draws or ties will not count at all 


flow of round
player clicks roll die button 

game logic generates random roll value from 1 - 6 for both player and house

results of each roll value are displayed 

game logic compares values against each other to determine who won or if ties

score board updateds according to who won the round 

game logic checks if either player reached 10 wins, if so, then popup will state who won

if no winner yet, maybe a popup will tell player to roll again, or the lable will state roll again... (will need to figure this out)
